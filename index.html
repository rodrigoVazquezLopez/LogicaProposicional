Instrucciones

La calculadora esta diseñada para realizar el cálculo de de proposiciones lógicas utilizando tablas de verdad.

Para ingresar las proposiciones utilice los botones A - I mientras que las operaciones lógicas que se pueden realizar son las que se muestran en la siguiente tabla:

Prioridad Operador Operación
1 ¬ Negación. Requiere un operando
2 ∧ Conjunción. Requiere dos operandos
3 ∨ Disyunción. Requiere dos operandos
4 → Implicación. Requiere dos operandos
5 ↔ Bicondicional. Requiere dos operandos
6 ⊕ Disyunción exclusiva. Requiere dos operandos

Se pueden utilizar paréntesis para agrupar las operaciones, sin embargo, en caso no hacerlo el orden de evaluación de la expresión se realiza con base en la prioridad del operador definido en la columna prioridad, siendo la negación la operación con mayor prioridad y la disyunción exclusiva la operación con menor prioridad.

En caso de equivocarse a la hora de ingresar la expresión utilice el botón DEL para borrar los caracteres necesarios.

Para borrar el contenido pulse el botón AC.

Para visualizar el resultado y el proceso de evaluación de la expresión utilice el botón RES.

Tokenizer

La expresión ingresada se analiza con la finalidad de identificar los distintos elementos que la componen: proposiciones, operaciones y paréntesis. Como resultado del análisis, cada elemento que compone la expresión se separa en tokens que se almacenan de manera independiente junto con una etiqueta que identifica su tipo.

Shunting Yard Algorithm

Una vez que se verificó que la cadena ingresada es sintácticamente correcta, se procede a evaluarla. Por ello, la expresión originalmente ingresada en notación infija se debe convertir a notación prefija (también conocida como notación polaca inversa) con la finalidad de facilitar el cómputo. Una forma de realizar la conversión es utilizando el algoritmo Shunting Yard

El algoritmo de Shunting Yard es un algoritmo basado en pila (stack). Para mostrar su funcionamiento se presenta la salida y el contenido de la pila de las etapas intermedias.
